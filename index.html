javascript

发表于 2017-06-22

js中常用的几种输出方式：

  • alert(‘内容’)：在浏览器弹出框显示我们的内容

  • document.write(‘内容’)：在页面中输出显示我们的内容

  • console.log(‘内容’):在控制台输出我们的内容，最常用的一种方式

  • innerHTML/innerText:向指定元素中动态添加内容，但是innerText不能识别标签
  • console.dir(“”)：在控制台输出衣蛾对象的详细信息
  • console.table()：把一个数据（json）按照表格的形式输出，我们在查看结构的时候更加的清晰一些
  • confirm()：是我们浏览器中确认选择弹出层，点击确定返回的是true，点击取消返回false
  • console.error(‘报错’)：输出报错信息
  • console.warn(‘警告’)

    innerHTML和innerText的区别

  • firefox浏览器中不支持innerText，而innerHTML不存在浏览器兼容问题
  • innerHTML不可以可以添加文字，还可以添加html标签，而innerText不能识别有效的html标签的，他会把html也当做文字添加进去

    js的命名规范

    javascript是运行在浏览器的轻量级的脚本编程语言，主要分三大部分
  • ECMAScript：定义了js里面的命名规范，变量，数据类型，基本语法，操作语句等最核心的东西
  • DOM：document object model 文档对象模型
  • BOM：browser object model 浏览器对象模型

    ECMAScript之js的命名规范

  • js中严格区分大小写
  • 使用驼峰命名法
    • 首字母小写，其余的每个有意义的单词的首字母大写 例personInfo
      • 可以使用数字（数字不能放在首位）、字母、下划线、$
  • 不能使用关键字和保留字
    • 关键字：在js中有特殊意义的字
    • 保留字：未来可能成为关键字的
